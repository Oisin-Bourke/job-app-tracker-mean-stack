Concept Its easy to lose track of your job applications. This application provides a simple way to keep track of all your job application data.
Functionality The application has the following functionality: Register with username and password. Login with your credentials. Add new jobs applications. Add notes to your job applications. Edit and delete your job applications. Export your job record to CSV for Excel spreadsheets.
Development The application is built using the MEAN (MongoDB, ExpressJS, Angular, NodeJS) stack - a modern open source development framework that allows the coder to use JavaScript on both the frontend and backend, while seamlessly transferring data from one layer to another in JSON format. Application Architecture Components Angular Angular frontend framework for building Single Page Applications in TypeScript. NodeJS NodeJS is a server-side JavaScript used to implement the REST API. Express Express is a web application framework for NodeJS that provides basic web-server creation and routing tools. Mongoose Mongoose is Object Data Modelling library for MongoDB and NodeJS that provides for schema creation and simplified data transfer. Atlas MongoDB Cloud-based version of MongoDB - the NoSQL document-oriented database that stores data in JSON-like documents. Angular Material Angular Material is a user interface library for quickly developing consistent and attractive web applications. JWT

Jason Web Toekns (JWT) are a standardised optionally validated and/or encrypted container format that is used to securely transfer information between two parties.

In this application, JWT are used to authenticate users. If a user provides the correct login credentials, a token, which includes a unique ID and role, is returned and stored in local storage. The token enables access to restricted routes.
