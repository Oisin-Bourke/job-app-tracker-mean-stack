Concept Its easy to loose track of your job applications. This app provides a simple way to keep track of all your job application data. Functionality The application provides the user with the following functionality. Register with username and password. Login with your credentials. Add new jobs applications. Add notes to your job applications. Edit and delete your job applications. Export your job record to CSV for Excel spreadsheets. Development The application is built using the MEAN (MongoDB, ExpressJS, Angular, NodeJS) stack - a modern open source development framework that allows the coder to use JavaScript on both the frontend and backend, while seamlessly transferring data from one layer to another in JSON format. Application Architecture Key Features Angular Angular frontend framework for building Single Page Applications in TypeScript. NodeJS NodeJS is a server-side JavaScript used to implement the REST API. Express Express is a web application framework for NodeJS that provides basic web-server creation and routing tools. Mongoose Mongoose is Object Data Modelling library for MongoDB and NodeJS that allows for schema creation and makes transferring data much easier. Atlas MongoDB Cloud-based version of MongoDB - the NoSQL document-oriented database that stores data in JSON-like documents.
